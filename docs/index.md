# Market Research Copilot

This is the documentation homepage for the Market Research Copilot project.

## About

The project automates market research by:

- parsing user queries,
- collecting search and news intelligence,
- scraping article content,
- generating structured reports with an LLM.

## Run locally

Use the instructions in the repository README to install dependencies and run the project.

## GitHub Pages

If GitHub Pages is enabled for the `docs` folder, this page will serve as your project site.
